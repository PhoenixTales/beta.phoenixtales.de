Contrib | Phoenix Tales

Do you want to contribute to PHOENIX ('Project Nyx')? We're looking for...

Engine Programmer: You will be responsible for implementing new features and reconstruct old functions (experience with C++ required), see our Git.

Scripters for ingames cutscenes: You build cutscenes according to given scripts; most likely with LeGo.

Animator: You'll be responsible to create and implement human animations and specific object interactions according to given instructions.

Comic & Video artist: You will work on spin-offs to our main project and realize teasers, trailers, intros, outros etc.

Graphic Artists: We need you to create low-res textures present in alpha versions of GOTHIC; usually not more than 512x512 pixels.

If you think you could help us in any other way, give us any kind of support or if you have something that we should add to the Gothic Archive, please reach out:

Email: mail@phoenixtales.de
